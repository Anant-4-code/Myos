QUESTION:
Write a Python program to create a class that acts as an iterator and generates factorial series up to n.

CODE:

class Factorial:
    def __init__(self, n):
        self.n = n
        self.i = 1
        self.fact = 1

    def __iter__(self):
        return self

    def __next__(self):
        if self.i > self.n:
            raise StopIteration
        self.fact *= self.i
        self.i += 1
        return self.fact

n = int(input("Enter n: "))

for i in Factorial(n):
    print(i)

QUESTION:
Create iterator class for factorial series.

CODE:

class Fact:
    def __init__(self,n):
        self.n=n
        self.i=1
        self.f=1

    def __iter__(self):
        return self

    def __next__(self):
        if self.i>self.n:
            raise StopIteration
        self.f*=self.i
        self.i+=1
        return self.f

for i in Fact(int(input())):
    print(i)
